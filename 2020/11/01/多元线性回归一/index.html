2.多元线性回归模型的基本假设

(1.) 解释变量 $x_1,x_2,\cdots,x_p$ 是确定变量，不是随机变量，且要求 $rank(X)=p+1<n$。这表明设计矩阵 $X$ 中的自变量列之间不相关，$X$ 是列满秩矩阵。

(2.) 随机误差项具有 $0$ 均值和等方差，即

该假定称为 $Gauss-Markov$ 条件。

(3.) 正态分布条件假定：

即 $\varepsilon\sim N(0,\sigma^2 I_n)$

在该条件下，我们有：

$\quad i.)\ y \sim N(X\beta,\sigma^2I_n)$

$\quad ii.)\ E(y)=X\beta$

$\quad iii.)\ Var(y)=\sigma^2I_n$

$\quad$

$proof.(tr(AB)=tr(BA))$

$\qquad tr(AB)=\sum\limits_{i=1}^n(AB)_{ii}$，其中 $(AB)_{ii}$ 表示 $AB$ 矩阵第 $i$ 行上主对角线元素的值。

$\qquad$而 $(AB)_{ii}$ 又等于 $A$ 的第 $i$ 行与 $B$ 的第 $j$ 列元素对应相乘，故 $(AB)_{ii}=\sum\limits_{j=1}^n a_{ij}b_{ji}.$

$\qquad$所以，$tr(AB)=\sum\limits_{i=1}^n(\sum\limits_{j=1}^n a_{ij}b_{ji})=\sum\limits_{j=1}^n(\sum\limits_{i=1}^n b_{ji}a_{ij})=tr(BA)$

$\blacksquare$

对于残差向量 $e=(e_1,e_2,\cdots,e_n)’=y-\hat{y}=(I-H)y$

我们有相关矩阵：

$
D(e)=Cov(e,e)
$
$\quad\quad\ \, =Cov((I-H)y,(I-H)y)$
$\quad\quad\ \, =(I-H)Cov(y,y)(I-H)’$
$\quad\quad\ \, =(I-H)\sigma^2I_n(I-H)’$
$\quad\quad\ \, =\sigma^2(I-H)(I-H)’$

注意到：

$(I-H)’=(I-X(X’X)^{-1}X’)’=I-X(X’X)^{-1}X’$

$(I-H)^2=(I-I-X(X’X)^{-1}X’)(I-I-X(X’X)^{-1}X’)$
$\qquad\quad\ \ \ \ = I-X(X’X)^{-1}X’-X(X’X)^{-1}X’+X(X’X)^{-1}X’$
$\qquad\quad\ \ \ \ =I-X(X’X)^{-1}X’=I-H$

故：

$D(e)=\sigma^2(I-H)(I-H)’=\sigma^2(I-H)^2=\sigma^2(I-H)$

从而：

下面我们考虑：

$E(\sum\limits_{i=1}^n e_i^2)=\sum\limits_{i=1}^n E(e_i^2)=\sum\limits_{i=1}^n [E(e_i^2)-(E(e_i))^2]=\sum\limits_{i=1}^n D(e_i)=(n-p-1)\sigma^2$

我们用 $e_i^2$ 代替 $E(e_i^2)$，得到 $\sigma^2$ 的估计值：

$\hat{\sigma^2}=\dfrac{1}{n-p-1}SSE=\dfrac{1}{n-p-1}\sum\limits_{i=1}^n e_i^2$

III 参数估计量的性质

性质 1.

$\hat{\beta}$ 是随机向量 $y$ 的一个线性变换。

注意到：$\hat{\beta}=(X^TX)^{-1}X^Ty$，所以 $\hat{\beta}$ 是随机向量 $y$ 的一个线性变换。

性质 2.

$\hat{\beta}$ 是 $\beta$ 的无偏估计。

$E(\hat{\beta})=E((X^TX)^{-1}X^Ty)$
$\qquad\ \ =(X^TX)^{-1}X^TE(y)$
$\qquad\ \ =(X^TX)^{-1}X^T(X\beta)\qquad(基本假设 2)$
$\qquad\ \ =(X^TX)^{-1}(X^TX)\beta$
$\qquad\ \ =\beta$

所以 $\hat{\beta}$ 是 $\beta$ 的无偏估计。

性质 3.

$D(\hat{\beta})=D((X^TX)^{-1}X^Ty)$
$\qquad\ \ =(X^TX)^{-1}X^TCov(y,y)((X^TX)^{-1}X^T)^{-1}$
$\qquad\ \ =(X^TX)^{-1}X^T\sigma^2I_nX(X^TX)^{-1}$
$\qquad\ \ =\sigma^2(X^TX)^{-1}$

性质 4.

预测函数 $\hat{y_0}=\hat{\beta_0}+\hat{\beta_1}x_{10}+\hat{\beta_2}x_{20}+\cdots+\hat{\beta_p}x_{p0}$ 是 $\hat{\beta}$ 的线性函数。

$Gauss-Markov$ 定理：

在假定 $E(Y)=X\beta,D(y)=\sigma^2I$时，$\beta$ 的任一线性函数 $c’\beta$ 的最小方差线性无偏估计(BLUE) 为 $c’\hat{\beta}$，其中，$c$ 是任一维常数向量，$\hat{\beta}$ 是 $\beta$ 的 $OLSE$.

对于上述定理有如下几条解释与注意：

(1) 若 $c_j$ 表示的是第 $j$ 个分量为 $1$，其余分量为 $0$ 的常向量，则 $Gauss-Markov$ 定理表明，$\beta_j$ 的最小方差线性无偏估计即为其 $OLSE$ 下的 $\hat{\beta_j}$

(2) 可能存在 $c’\beta$ 的非线性估计 $\widetilde{\beta}$，一方面它是无偏估计，另一方面它的方差更小。

(3) 可能存在 $c’\beta$ 的有偏估计，在某种意义下比 $OLSE$ 更加好。

(4) 在基本假设 2 下，$c’\hat{\beta}(OLSE)$ 是 $c’\beta$ 的最小方差无偏估计。即不管 $c’\beta$ 的线性估计还是非线性估计，在无偏性与方差意义下，$OLSE$ 是 $c’\beta$ 的最佳估计。

$proof.$

设 $\alpha^TY$ 是 $c’\beta$ 的任意一线性无偏估计，根据其无偏性，我么有：对于任意的 $\beta$

从而有 $c’=\alpha^TX$.对于任意一个向量，我们定义$||Z||^2=Z^TZ$，从而 $||Z||^2=0$，当且仅当 $||Z||=0.$

下面我们来计算其方差：

$
Var(\alpha^TY)=\alpha^TVar(Y)\alpha
$
$\qquad\quad\quad\ \ \ \, =\sigma^2||\alpha||^2$
$\qquad\quad\quad\ \ \ \, =\sigma^2||\alpha-X(X^TX)^{-1}c+X(X^TX)^{-1}c||^2$
$\qquad\quad\quad\ \ \ \, =\sigma^2||\alpha-X(X^TX)^{-1}c||^2+\sigma^2||X(X^TX)^{-1}c||^2+2\sigma^2c^T(X^TX)^{-1}X^T(\alpha-X(X^TX)^{-1}c)$

其中

$\sigma^2||X(X^TX)^{-1}c||^2=\sigma^2(c’(X^TX)^{-1}X^T)(X(X^TX)^{-1}c)$
$\qquad\qquad\qquad\qquad\ =\sigma^2c’(X^TX)^{-1}c$

$2\sigma^2c^T(X^TX)^{-1}X^T(\alpha-X(X^TX)^{-1}c)$
$=2\sigma^2\alpha^T X(X^TX)^{-1}X^T(\alpha-X(X^TX)^{-1}X^T\alpha)$
$=2\sigma^2\alpha^T X(X^TX)^{-1}X^T\alpha-2\sigma^2\alpha^T X(X^TX)^{-1}X^TX(X^TX)^{-1}X^T\alpha$
$=2\sigma^2\alpha^T X(X^TX)^{-1}X^T\alpha-2\sigma^2\alpha^T X(X^TX)^{-1}X^T\alpha=0$

所以

$
Var(\alpha^TY)=\sigma^2||\alpha-X(X^TX)^{-1}c||^2+\sigma^2c’(X^TX)^{-1}c
$

注意到 $Var(c’\hat{\beta})=c’\sigma^2(X^TX)^{-1}c$，所以 $Var(c’\hat{\beta})\le Var(\alpha^TY)$，且等号成立当且仅当 $\alpha=X(X^TX)^{-1}c$，即 $\alpha^TY=c’(X^TX)^{-1}X^TY=c’\hat{\beta}$

$\blacksquare$

$proof.$

$Cov(\hat{\beta},e)$
$=Cov((X^TX)^{-1}X^Ty,(I-H)y)$
$=\sigma^2(X^TX)^{-1}X^T(I-H)$
$=\sigma^2(X^TX)^{-1}X^T-(X^TX)^{-1}X^TX(X^TX)^{-1}X^T)$

性质 6.

当 $y\sim N(\beta,\sigma^2I_n)$ 时，则
(1) $\hat{\beta}\sim N(\beta,\sigma^2(X’X)^{-1})$
(2) $\dfrac{SSE}{\sigma^2}\sim \chi^2(n-p-1)$
(3) $\hat{\beta}$ 与 $\dfrac{SSE}{\sigma^2}$ 相互独立。

$\eta=(\eta_1,\eta_2,\cdots,\eta_n)’=P^T\varepsilon$

故 $\dfrac{SSE}{\sigma^2}=\dfrac{\eta_1}{\sigma^2}+\cdots+\dfrac{\eta_{n-p-1}}{\sigma^2}\sim \chi^2(n-p-1)$
