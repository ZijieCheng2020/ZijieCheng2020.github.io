III 回归方程的显著性检验

1. $t$ 检验

目的：$t$ 检验是用于检验回归系数的显著性。

步骤：

$Step1.$ 建立检验假设。
原假设 $H_0:\beta_1=0$, 备择假设：$H_1:\beta_1\ne0$

$Step2.$ 构造检验统计量。
由于 $\hat{\beta_1}\sim (\beta_1，\dfrac{\sigma^2}{L_{xx}})$，所以当 $H_0:\beta_1=0$ 成立时有：$\hat{\beta_1}\sim (0，\dfrac{\sigma^2}{L_{xx}})$
构造 $t$ 统计量：

$Step3.$ 根据所给数据计算 $P$ 值，并给出检验结果。

2. $F$ 检验

目的：$F$ 检验是根据平方和分解式，直接从回归效果检验回归方程的显著性。

平方和分解式 (注意：无截距项时不一定成立) ：

其中：
总的离差平方和：$SST=\sum\limits_{i=1}^n(y_i-\bar{y})^2$
回归平方和：$SSR=\sum\limits_{i=1}^n(\hat{y_i}-\bar{y})^2$
残差平方和：$SSE=\sum\limits_{i=1}^n(y_i-\hat{y_i})^2$
所以分解式可简记如下：

$Proof.$

$\sum\limits_{i=1}^n(y_i-\bar{y})^2=\sum\limits_{i=1}^n(y_i-\hat{y_i}+\hat{y_i}-\bar{y})^2$
$\qquad\qquad\quad\ =\sum\limits_{i=1}^n[(y_i-\hat{y_i})^2+(\hat{y_i}-\bar{y})^2+2(y_i-\hat{y_i})(\hat{y_i}-\bar{y})]$

其中：

$\sum\limits_{i=1}^n(y_i-\hat{y_i})(\hat{y_i}-\bar{y})=\hat{\beta_1}\sum\limits_{i=1}^nx_i(y_i-\hat{y_i})+(\beta_0-\bar{y})\sum\limits_{i=1}^n(y_i-\hat{y_i})$

注意到：

$\sum\limits_{i=1}^ne_i=0,\sum\limits_{i=1}^nx_ie_i=0$

所以：

$\sum\limits_{i=1}^n(y_i-\hat{y_i})(\hat{y_i}-\bar{y})=0$

综上:平方分解和式成立。

3.相关系数 $r$ 检验

目的：检验相关系数 $r$ 的显著性，即通过 $r$ 的大小来衡量变量 $x$ 与 $y$ 之间的线性相关程度。

构造相关系数：

4.三种检验的关系：

对于一元线性回归而言，这三种检验的结果是完全一致的。
即

$(1)\ t=\dfrac{\hat{\beta_1}\sqrt{L_{xx}}}{\hat{\sigma}}=\dfrac{\sqrt{n-2}\ r}{\sqrt{1-r^2}}$

$(2)\ F=\dfrac{SSR/1}{SSE/(n-2)}=\dfrac{\hat{\beta_1}^2\cdot L_{xx}}{\hat{\sigma}^2}=t^2$

$proof.$

$(1)$ 注意到：

$\hat{\sigma}^2=\dfrac{1}{n-2}\sum\limits_{i=1}^n(y_i-\hat{y_1})^2$

$\ \ \ \ \ =\dfrac{1}{n-2}\sum\limits_{i=1}^n(L_{yy}-\sum\limits_{i=1}^n(\hat{y_i}-\bar{y})^2)$

$\ \ \ \ \ =\dfrac{1}{n-2}(L_{yy}-\sum\limits_{i=1}^n(\hat{\beta_1}x_i+\hat{\beta_0}-(\hat{\beta_0}+\hat{\beta_1}\bar{x}))^2)$

$\ \ \ \ \ =\dfrac{1}{n-2}(L_{yy}-\dfrac{L_{xy}^2}{L_{xx}^2}\sum\limits_{i=1}^n(x_i-\bar{x})^2)$

$\ \ \ \ \ =\dfrac{1}{n-2}(L_{yy}-\dfrac{L_{xy}^2}{L_{xx}})$

故有：

$t=\dfrac{\hat{\beta_1}\sqrt{L_{xx}}}{\hat{\sigma}}=\dfrac{\sqrt{n-2}\hat{\beta_1}\sqrt{L_{xx}}}{\sqrt{L_{yy}-\dfrac{L^2_{xy}}{L_{xx}}}}$

$\ \ =\dfrac{\sqrt{n-2}\hat{\beta_1}\sqrt{\dfrac{L_{xx}}{L_{yy}}}}{\sqrt{1-\dfrac{L_{xy}^2}{L_{xx}L_{yy}}}}$

$\ \ =\dfrac{\sqrt{n-2}\ r}{\sqrt{1-r^2}}$

$(2)$ 由 $(1)$ 得：

$SSR=\dfrac{L_{xy}^2}{L_{xx}}，SSE=L_{yy}-\dfrac{L_{xy}^2}{L_{xx}}$

所以：

$F=\dfrac{SSR/1}{SSE/(n-2)}$

$\quad=(n-2)\dfrac{\dfrac{L_{xy}^2}{L_{xx}}}{L_{yy}-\dfrac{L_{xy}^2}{L_{xx}}}$

$\quad =(n-2)\dfrac{r^2}{1-r}$

$\quad=t^2$

$\quad=\dfrac{\hat{\beta_1}^2\cdot L_{xx}}{\hat{\sigma}^2}$

5. 决定系数

定义：回归平方和 $(SSR)$ 与总离差平方和 $(SST)$ 之比定义为决定系数，即回归平方和在总离差平方和中的占比，记为 $r^2$：

$r^2=\dfrac{SSR}{SST}=\dfrac{L^2_{xy}}{L_{xx}L_{yy}}=(r)^2$

意义：决定系数 $r^2$ 是一个反应直线与样本观测拟合优度的相对指标，将残差 $(e_i=y_i-\hat{\beta_0}-\hat{\beta_1}x_i)$ 作为误差 $(\varepsilon_i=y_i-\beta_0-\beta_1x_i)$ 的估计，我们认为 $r^2$ 越大，残差占比越小，拟合优度越好。

IV. 残差分析

1. 残差的基本性质

$(i.)\ E(e_i)=0$

$proof.$

$E(e_i)=E(y_i)-E(\hat{y_i})$
$\quad\quad \ \ \ =(\beta_0+\beta_1x_i)-(\beta_0+\beta_1x_i)=0$

$(ii.)\ Var(e_i)=[1-\dfrac{1}{n}-\dfrac{(x_i-\bar{x})^2}{L_{xx}}]\sigma^2$

$(iii.)$ 残差满足约束条件：$\sum e_i=0,\sum x_ie_i=0$，即残差是相关的，不独立。

V. 回归系数的区间估计

$t=\dfrac{(\hat{\beta_1}-\beta_1)\sqrt{L_{xx}}}{\sigma}\sim t(n-2)$

$P(|\dfrac{(\hat{\beta_1}-\beta_1)\sqrt{L_{xx}}}{\sigma}|<t_{\alpha/2}(n-2))=1-\alpha$
